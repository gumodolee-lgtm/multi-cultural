# 경쟁/참고 서비스 상세 분석
> 조사일: 2026-03-12
> 출처: AI 학습 데이터 기반 (지식 기준 ~ 2025년 8월) + 연구 에이전트 분석
> ⚠️ 실제 개발 착수 전 각 포털에 직접 접속하여 현재 상태를 재확인할 것

## 핵심 결론

**Windows 데스크탑 다문화 정보 앱: 국내 전무 (블루오션 확인)**

모든 기존 서비스는 웹 기반이며, 일부만 Android/iOS 앱 제공.
데스크탑 전용 앱은 단 하나도 존재하지 않는다.

---

## 1. 다누리 (danuri.go.kr)
- **운영**: 여성가족부 / 한국건강가정진흥원
- **특징**: 결혼이민자 특화, 13개국어 지원
- **API**: 없음 (RSS 불안정)
- **갭**: 이주 노동자·유학생·난민 제외, 실시간 업데이트 부족

## 2. 하이코리아 (hikorea.go.kr)
- **운영**: 법무부 출입국·외국인정책본부
- **특징**: 비자·체류 행정 처리, KIIP 관리
- **API**: data.go.kr 경유 통계 일부
- **갭**: 4개 언어, UI 복잡, 법령 문어체

## 3. 법제처 (law.go.kr) ★ API 활용 가능
- **운영**: 법제처
- **특징**: 모든 현행 법령 원문, 개정이력 완전
- **API**: 공식 REST API 제공 (법령 조회, 목록, 상세)
- **입법예고 RSS**: 공식 제공
- **갭**: 일반인 이해 어려움, 생활 정보 연결 없음

## 4. 공공데이터포털 (data.go.kr) ★ API 활용 가능
- **운영**: 행정안전부
- **주요 데이터셋**:
  - 외국인 주민 현황 통계 (행안부)
  - 다문화 학생 현황 (교육부)
  - 출입국 외국인 정책 통계월보 (법무부)
  - 다문화가족 실태조사 (여가부)
  - 사회통합프로그램 이수자 현황 (법무부)
  - 외국인근로자 고용 현황 (고용노동부)
  - 다문화가족지원센터 현황 (여가부)
- **API**: REST API (JSON/XML), API 키 무료 발급
- **갭**: 원시 집계 데이터, 갱신 불규칙, UI 빈약

## 우리 프로그램의 차별화 포인트

1. **유일한 Windows 데스크탑 앱** - 브라우저 없이 바로 실행
2. **통합 뷰어** - 5개+ 사이트 정보를 1개 앱으로
3. **오프라인 캐시** - 인터넷 없어도 최근 데이터 열람
4. **스마트 알림** - 법령 개정, 마감 임박 Windows 토스트 알림
5. **다국어 번역** - 한국어 법령을 베트남어 등으로 실시간 번역
6. **AI 요약** - 복잡한 법령을 쉬운 말로 요약
7. **지역 필터** - 내 지역 지원사업만 표시
8. **내보내기** - Excel/PDF로 보고서 작성 지원
